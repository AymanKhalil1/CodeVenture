Young Dragons - Level 1 : ScratchJr

Recommended age:

5-8 years old

Duration:

3 Hours (4 * 45 minutes)

Course Summary:

This is a novice level course for children of ages 5-8. The course utilizes a block based programming platform, ScratchJr, and gets fresh students familiarized with its basic controls through simple animations.

Requirements

  • Coding Setup:
    • Android Tablet / Apple iPad.
    • ScratchJr app(free)
  • Lesson Setup:
    • A computer running Zoom
    • Microphone
    • Speakers
    • Headphones(optional)
    • Webcam(optional)
  • Other Requirements
    • A parent/guardian needs to be present during the online sessions.
  • Prerequisites:
    • No prior coding experience required

Smart Kangaroos - Level 1 : Scratch

Recommended age:

  • New Student: 8-9 years.
  • Young Dragons - Level 2 graduate: any age.

Duration

6 Hours (6 * 60 minutes)

Course Summary:

This is a novice level course that aims to teach children basic programming concepts using the versatile and entertaining Scratch platform. All the concepts are eventually applied into making an easy video game.

Requirements

  • Coding Setup:
    • A computer running Windows (recommended) or MacOS, with a web browser installed.
    • A free Scratch account is required for this course. Sign up on the official Scratch platform website. Click the Join Scratch link at the top of the page.
  • Lesson Setup:
    • A computer running Zoom
    • Microphone
    • Speakers
    • Headphones(optional)
    • Webcam(optional)
  • Other Requirements
    • A parent/guardian needs to be reachable during the online sessions.
  • Prerequisites:
    • Some computer skills are needed: web browsing, switching between active windows, typing on the keyboard, using the mouse.
    • No prior coding knowledge is required.

Flying Eagles Level 1 : Python

Recommended age:

10-12 years old

Duration

8 Hours (8 * 60 minutes)

Course Summary:

This is a novice level course for children of ages 10-12. It introduces basic programming concepts, including variables, flow control, loops, and lists. The programming language used is Python. The course also demonstrates the application of some of these concepts through the use of the Turtle graphics Module in Python.

Requirements

  • Coding Setup:
    • A computer running Microsoft Windows (recommended) / MacOS
    • Python 3
  • Lesson Setup:
    • A computer running Zoom
    • Microphone
    • Speakers / headphones
    • Webcam(optional)
  • Other Requirements
    • A parent/guardian needs to be reachable during the online sessions.
  • Prerequisites:
    • Some computer skills are needed: web browsing, switching between active windows, typing on the keyboard, using the mouse.
    • No prior coding experience required

Flying Eagles - Level 2 : Python

Recommended age:

10-12 years old

Duration

7 Hours (7 * 60 minutes)

Course Summary:

This is an intermediate level course for students of ages 10-12 years old. The course introduces more advanced programming concepts to get students familiarized with the structures of real life programs. Functions and Object Oriented Programming are among the concepts explored throughout the course. Finally, students are led in an instructor walkthrough to make a video game, in the style of the classic video game 'Pong', using the Pygame module.

Requirements

  • Coding Setup:
    • A computer running Microsoft Windows (recommended) / MacOS
    • Python 3
  • Lesson Setup:
    • A computer running Zoom
    • Microphone
    • Speakers / headphones
    • Webcam(optional)
  • Other Requirements
    • A parent/guardian needs to be reachable during the online sessions.
  • Prerequisites:
    • Student must have successfully completed the Flying Eagles - Level 1 course.
