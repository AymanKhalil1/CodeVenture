Young Dragons Level 1

Course Name:

Young Dragons: Level 1

Category:

Young Dragons

Recommended Age

5-8 years old

Duration

3 Hours (4*45 mins)

Course Summary:

This is a novice level course for children of ages 5-8. The course utilizes a block based programming platform, ScratchJr, and gets fresh students familiarized with its basic controls through simple animations.

Requirements:

  • Coding Setup:
    • Android Tablet / Apple iPad.
    • ScratchJr app. (free)
  • Lesson Setup:
    • A computer running Zoom.
    • Microphone.
    • Speakers.
    • Headphones. (Optional)
    • Webcam. (Optional)
  • Other Requirements:
    • A parent/guardian needs to be present during the online lessons.
  • Prerequisites:
    • No prior coding exposure required

Flying Eagles level 1

Course Name:

Flying Eagles: Level 1

Category

Flying Eagles

Recommended Age:

9-12 years old

Duration:

8 Hours (8 * 60 mins)

Course Summary:

This is a novice level course for children of ages 9-12. It introduces basic programming concepts, including variables, flow control, loops, and lists. The programming language used is Python. The course also demonstrates the application of some of these concepts through the use of the Turtle graphics Module in Python.

Requirements:

  • Coding Setup:
    • PC running Microsoft Windows (Recommended) / Mac OS / Linux
    • Python 3
  • Lesson Setup:
    • A computer running Zoom.
    • Microphone.
    • Speakers.
    • Headphones. (Optional)
    • Webcam. (Optional)
  • Other Requirements:
    • A parent/guardian needs to be reachable during the online lessons.
  • Prerequisites
    • No prior coding exposure required
